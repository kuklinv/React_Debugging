# react-contacts

A sample React app created to teach React DevTools debugging

## How to run the app

Clone the repository or download the project. Then, install dependencies:
```bash
npm install
# --- or ---
yarn install
```

Run the app with the following command:
```bash
npm start
# --- or ---
yarn start
```
